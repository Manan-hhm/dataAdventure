<h3>Data Science</h3>
Doing any sort of scientific studies/analysis on data. <br>
Processes of data science include experiments, collection, cleaning, analyzing data, making conclusions and more.

<h3>Why do data science?</h3>
<ul>
  <li>Great career prospects<br>
      -> From startups to MAANG like companies, oil companies, retail brands, unicorns.</li>
  <li>Good work-life balance<br>
      -> Output is not a program but model/analysis that cannot break all of a sudden.</li>
</ul>

<h3>Steps of Data Science</h3> 
Data Science process involves a lot of steps like data scraping, fetching, processing, cleaning, analyzing, presenting, visualization, ML modelling, deployment, scaling, etc.

<h3>Programming Languages</h3>
Two significant programming languages in Data Science: 
<ul>
  <li><b>Python: </b> 
    <br> -> Most widely used in Data Science.
    <br> -> Most of the problems need it for solutions. </li>
  <li><b>R: </b> 
    <br> -> Used by a few teams/organizaions.
    <br> -> Statistics-oriented language for analysis and research. 
    <br> -> Not friendly for deployment and industry focused work.</li>
</ul>

<h3>Python libraries</h3>
These libraries are used for various data science processes. They include: 
<ul>
  <li>Numpy: Matrix & Tensor operations.</li>
  <li>Pandas: Reading, cleaning and data operations.</li>
  <li>OpenCV: Reading & basic operations on images.</li>
  <li>Matplotlib & seaborn: For data visualization.</li>
</ul>
